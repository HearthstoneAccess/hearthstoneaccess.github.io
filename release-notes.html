Release Notes

October 1 2021: 21.3.0.93849-10

  • You can now play ranked, casual and friendly matches against other players once you've unlocked all classes. Online matches always run in full speed and have a per-turn time limit so make sure you are comfortable with game speed 10 against AI first
  • The social menu is now accessible through the F4 key. You can use it to invite, message, spectate and challenge your Battle.net friends
  • Player names and classes can now be read for both Heroes when playing online
  • The in-game emotes menu can now be accessed by pressing Space when selecting your Hero. Squelching is also available when selecting your Opponent's Hero
  • Added 15 and 5 seconds remaining warnings when playing online games
  • Decks can now be exported in the edit deck menu
  • Rewards Track XP changes and level ups are now read
  • Wild and Classic modes are now available to players who have reached Bronze 10
  • Chinese is now fully available thanks to the awesome contributions of oldman

September 21 2021: 21.3.0.93849-9

  • Updated to Hearthstone 21.3

September 17 2021: 21.2.0.91456-8

  • Book of Heroes Uther is now available
  • Opening card packs is now accessible
  • The tradeable game mechanic is now accessible. Tradeable cards can be traded using T
  • The original stats of minions as well as any enchantments they may have can now be read by pressing K when hovering a minion. You can use this key whenever you need to find out which minions need healing or when you don't know who's buffing a particular minion
  • Chinese is now available. Note that a few translations that were introduced in the latest patches are still missing and will default to English
  • Various bug fixes and accessibility enhancements

September 2 2021: 21.2.0.91456-7

  • Hotfix the new flow which was preventing new players from accessing the tutorial after the latest patch
  • Remove unnecessary visual tutorials after players finish apprenticeship which were causing problems to players using OCR
  • Make the claim your free deck reward from completing apprenticeship accessible

August 31 2021: 21.2.0.91456-6

  • Updated everything to Hearthstone 21.2 which revamped the main menu interface
  • Book of Heroes Garrosh is now available
  • Quests, sidequests and questlines are now accessible
  • Enemy secrets and quests can now be read using Shift+S and the arrow keys
  • Significant user experience improvements to players using Microsoft Speech API as their screenreader
  • Pushing the end turn button when minions can still attack or the Hero Power can still be used now specifically warns the player
  • Minor translation fixes

August 27 2021: 21.0.3.91040-5

  • Fixed issue causing Hero cards such as Lord Jaraxxus to be incorrectly read
  • Added Home, End, Page Up and Page Down keys to the My Collection screen
  • Minor fixes in all localized versions of the game

August 26 2021: 21.0.3.91040-4

  • The first version of the My Collection screen including standard deck building is now available
  • The main game options menu is now fully accessible so players can adjust music volume, graphics resolution, graphics quality and others
  • Polish, European Spanish, Mexican Spanish and Brazilian Portuguese are now available thanks to the awesome contributions of Nuno, Alisson, pildain, AAyoubelbakhti and sanslash332
  • Added additional defensive code and error messages to help track down a rare bug with Choose One cards
  • Minor bug fixes

August 17 2021: 21.0.3.91040-3

  • Released Localization support so users can contribute translations for different languages
  • Book of Heroes Rexxar is now available
  • Mulligan keys changed. Space now marks cards for replacement and Enter confirms them
  • Some menus such as the choose deck menu are now horizontal so the up and down arrow keys can be used to read additional information such as Hero levels
  • The Home and End keys can now be used to quickly navigate to the first and last card in the zone that's currently being read
  • Damaging armored Heroes now differentiates between armor lost and damage taken
  • Added card rarity to all non-free cards at the end
  • Removed Hero triggered messages when mission events or animations happen
  • Fixed bug in Choose One cards such as Ethereal Conjurer which broke input

August 10 2021: 21.0.0.88998-2

  • Book of Heroes Jaina is now available
  • Numbers in the cards such as attack and health are now read as digits to improve the experience of languages other than english

August 6 2021: 21.0.0.88998-1

  • Hearthstone Access was launched
