Release Notes

August 26 2021: 21.0.3.91040-4

  • The first version of the My Collection screen including standard deck building is now available
  • The main game options menu is now fully accessible so players can adjust music volume, graphics resolution, graphics quality and others
  • Polish, European Spanish, Mexican Spanish and Brazilian Portuguese are now available thanks to the awesome contributions of Nuno, Alisson, pildain, AAyoubelbakhti and sanslash332
  • Added additional defensive code and error messages to help track down a rare bug with Choose One cards
  • Minor bug fixes

August 17 2021: 21.0.3.91040-3

  • Released Localization support so users can contribute translations for different languages
  • Book of Heroes Rexxar is now available
  • Mulligan keys changed. Space now marks cards for replacement and Enter confirms them
  • Some menus such as the choose deck menu are now horizontal so the up and down arrow keys can be used to read additional information such as Hero levels
  • The Home and End keys can now be used to quickly navigate to the first and last card in the zone that's currently being read
  • Damaging armored Heroes now differentiates between armor lost and damage taken
  • Added card rarity to all non-free cards at the end
  • Removed Hero triggered messages when mission events or animations happen
  • Fixed bug in Choose One cards such as Ethereal Conjurer which broke input

August 10 2021: 21.0.0.88998-2

  • Book of Heroes Jaina is now available
  • Numbers in the cards such as attack and health are now read as digits to improve the experience of languages other than english

August 6 2021: 21.0.0.88998-1

  • Hearthstone Access was launched
